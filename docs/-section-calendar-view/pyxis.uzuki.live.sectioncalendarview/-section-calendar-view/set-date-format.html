open fun setDateFormat(dateFormat: String): Unit

Sets the dateFormat of the currently displayed month. Default is yyyy MMM 위에 표시되는 현재 보여지고 있는 달의 날짜 포맷을 설정합니다. 기본적으로 yyyy MMM이 사용됩니다.

The actual display format changes depending on which language setting the current device is in. 현재 기기가 어떤 언어 설정으로 되어있는지에 따라 실제 표시되는 형식이 변경됩니다.
